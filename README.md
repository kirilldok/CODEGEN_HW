# Code Generation

## Описание

Реализация бэкенда компилятора: преобразование оптимизированного промежуточного представления (Generic IR) в машинный код.

**Выполняемые этапы:**
1. Разрушение SSA-формы
2. Легализация инструкций
3. Выбор машинных инструкций
4. Распределение регистров (раскраска графа / линейный алгоритм)
5. Планирование инструкций (списочное планирование) для самого объемного базового блока

**Целевая архитектура:** RISC-V
