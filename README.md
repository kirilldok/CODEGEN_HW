Codegen Project — Вариант 1 (RISC-V)

Данный проект представляет собой реализацию бэкенда компилятора (Code Generation) для учебного курса «Введение в тензорные компиляторы».

Выполненные этапы:

Instruction Selection: Преобразование Generic IR (Вариант 1) в систему команд RISC-V (RV32I).

Register Allocation: Алгоритм раскраски графа для распределения физических регистров (x10-x17).

Instruction Scheduling: Алгоритм списочного планирования (List Scheduling) для минимизации простоев конвейера (с учетом латентности LW и MUL).

Структура проекта

include/ — Заголовочные файлы с описанием структур IR и алгоритмов.

src/ — Реализация логики:

main.cpp — Основной драйвер, собирающий пайплайны.

reg_alloc.cpp — Модуль распределения регистров.

scheduler.cpp — Модуль планирования инструкций.

test/ — Модульные тесты для проверки корректности алгоритмов.

CMakeLists.txt — Сценарий сборки.

Требования

Для сборки и запуска вам понадобятся:

CMake версии 3.10 или выше.

Компилятор C++ с поддержкой стандарта C++17 (GCC, Clang или MSVC).

Утилита Make (для Linux/macOS).

Описание алгоритмов

Selection: Инструкция result += 2 * 3 * a[j] * 5 - i - 2 оптимизирована с помощью Constant Folding в result += 30 * a[j] - i - 2. Используются инструкции SLLI (смещение), LW (загрузка), MUL и SUB.

RegAlloc: Строится граф интерференции переменных. Переменные, чьи интервалы жизни пересекаются, получают разные физические регистры xN.

Scheduling: Строится DAG (направленный ациклический граф) зависимостей по данным. Инструкции с большой задержкой (например, MUL — 3 цикла) планируются заранее, чтобы избежать stall-эффектов.
